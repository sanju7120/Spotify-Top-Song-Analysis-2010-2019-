# Spotify Music Popularity Trend Analysis (2010–2019) 
Project Overview:
This project analyzes Spotify music data from 2010 to 2019 to identify key factors influencing song popularity and understand long-term music trends.The analysis helps simulate real-world business scenarios such as improving recommendation systems, identifying successful musical features and understanding listener preferences.

Dataset:
Source: Kaggle
Time Period: 2010–2019
Records: ~1000 songs
Features: Artist, Song Name, Popularity, Danceability, Energy, Genre,Tempo

Analysis Performed:
Data cleaning and preprocessing
Feature correlation analysis
Trend analysis across years
Top artist and genre analysis
Distribution analysis of musical attributes

Key Insights:
Songs with higher danceability and energy were frequently present among top-ranked tracks.
Popularity showed noticeable variation across different years.
Certain artists consistently appeared in top charts.
Pop and Dance genres dominated streaming trends.

Business Impact:
The insights from this analysis can help music platforms:
Improve recommendation algorithms
Optimize promotional strategies
Support artist collaboration decisions
Enhance user engagement
